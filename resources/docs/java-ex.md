# How to read some Java expressions and statements

| Java expression | Statement |
|---|---|
| == | equal to |
| != | not equal |
| >= | greater than or equal to |
| <= | less than or equal to |
| ++ |Plus plus|
|--| Minus minus|
| /* |	Slash star|
| */ |	Star slash|
| -> |	Lambda token, Arrow|
| &&| 	And|
| \|\|  |	Or |
| .* | dot star|
| \n  | slash n |
| : :  | Colon Colon |
| java -jar  | java minus jar |
| sql | sikuel |
| \<?\> | wildcard |
